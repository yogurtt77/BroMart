# BROMART

Магазин онлайн-покупок учреждение №57

## Стек технологий

- React
- JavaScript
- SCSS
- React Router

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

## Страницы

- **Главная** (`/`) - Главная страница с каталогом категорий
- **Вопрос-ответ** (`/faq`) - Страница с часто задаваемыми вопросами
- **Жалобы и предложения** (`/complaints`) - Форма обратной связи
- **Вход** (`/login`) - Страница авторизации
- **Регистрация** (`/register`) - Страница регистрации
- **Корзина** (`/cart`) - Корзина покупок

## Примечание

SVG иконки и изображения товаров нужно добавить в папку `public/`.
